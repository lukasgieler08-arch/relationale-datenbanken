Teil C: Datum-Funktionen in Datenbankabfragen

Beispiele

SELECT *
FROM EINSATZ
WHERE Datum >= CURDATE();
SELECT EinsatzID, DATEDIFF(CURDATE(), Datum) AS TageSeitEinsatz
FROM EINSATZ;

Wichtige Funktionen

CURDATE(), NOW(), YEAR(), MONTH(), DATEDIFF(), DATE_ADD()

Merksatz: Zeitraumgrenzen immer eindeutig formulieren.
