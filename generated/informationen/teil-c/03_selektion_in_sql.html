Teil C: Selektion in SQL

Kernidee

Selektion filtert Zeilen über die WHERE-Klausel.

SELECT Name, Kategorie
FROM FOODTRUCK
WHERE Kategorie = 'Vegan' AND Rating >= 4.5;

Wichtige Operatoren

=, <>, >, >=, <, <=, IN, BETWEEN, LIKE, IS NULL

Merksatz: Klammern machen komplexe Bedingungen sicherer und klarer.
