## Schritt-für-Schritt: Suchbare HTML-Ansicht im Codespace öffnen

Die interaktive Suchfunktion steht nur in der HTML-Datei zur Verfügung. So öffnest du sie im Codespace:

**Voraussetzung:** Die Erweiterung *Live Server* muss installiert sein.
Sie ist im Codespace standardmäßig vorinstalliert. Falls nicht: Erweiterungen-Symbol in der linken Seitenleiste anklicken, nach „Live Server" suchen und installieren.

**Schritte:**

1. Den Explorer in VS Code öffnen (Symbol ganz oben links in der Seitenleiste, sieht aus wie zwei Seiten).
2. Im Explorer die Datei
   `generated/informationen/begrifflichkeiten/stichwortverzeichnis_relationale_datenbanken.html`
   suchen und mit einem **Rechtsklick** darauf klicken.
3. Im Kontextmenü den Eintrag **„Open with Live Server"** auswählen.
4. Der Browser öffnet sich automatisch mit der Suchseite.
5. Im Suchfeld oben einfach einen Begriff eingeben, z. B. `JOIN`, `3NF` oder `Anomalie`.
6. Die Tabelle filtert die Ergebnisse sofort live.

**Hinweis:** Wenn kein Browser-Fenster aufgeht, schaue unten in der VS-Code-Statusleiste –
dort erscheint nach dem Start von Live Server ein klickbarer Link (z. B. `Port 5500`).

**Alternative ohne Live Server:**

Falls Live Server nicht verfügbar ist, kannst du die Markdown-Version nutzen:
[Stichwortverzeichnis (Markdown)](./stichwortverzeichnis_relationale_datenbanken.md)
Dort sind alle Begriffe tabellarisch aufgelistet, aber ohne Suchfunktion.

## Didaktischer Zweck

Dieses Verzeichnis bündelt zentrale Begriffe aus BPE 3 (Relationale Datenbanken) in aufsteigender Sortierung.
Fokus: 3NF, Redundanzvermeidung, Integrität, SQL-Grundklauseln sowie sichere Anwendung in Klassenarbeiten und Übungen.
